- (Hash or nil) hit(target)

衝突判定をします。自身だけでなく、子と相手、あるいは子同士の判定も行われます。 結果は返り値、またはブロックで受け取る事も可能です。

Parameters:

  • target (Actro or Scene) —

    衝突相手となるアクターまたはシーン

Yields:

  • (Hash) —

    衝突があった場合、衝突情報と共に渡される。情報ハッシュの内容は以下、

    { 衝突した子要素１ => { 衝突部位インデックス => { 相手衝突部位インデックス => { 詳細情報 }} }, …}

Returns:

  • (Hash or nil) —

    衝突情報ハッシュまたは未衝突の際はnilを返す。
