Overview

- (Object) draw(image, x, y, opts = {})

イメージをスクリーン上にオプション付きで描画

Parameters:

  • image (Image) —

    イメージ

  • x (Numeric) —

    描画先のX座標

  • y (Numeric) —

    描画先のY座標

  • opts (Hash) (defaults to: {}) —

    オプション格納

Options Hash (opts):

  • :src_x (Fixnum) — default: 0 —

    描画するイメージのX位置

  • :src_y (Fixnum) — default: 0 —

    描画するイメージのY位置

  • :src_w (Fixnum) — default: Image#width —

    描画するイメージの幅

  • :src_h (Fixnum) — default: Image#height —

    描画するイメージの高さ

  • :dst_w (Fixnum) — default: #width —

    スクリーンの描画範囲の幅

  • :dst_h (Fixnum) — default: #height —

    スクリーンの描画範囲の幅の高さ

  • :angle (Fixnum) — default: 0 —

    イメージの回転角度

  • :scale_x (Float) — default: 1.0 —

    イメージの幅の拡大倍率

  • :scale_y (Float) — default: 1.0 —

    イメージの高さを拡大倍率

  • :center_x (Float) — default: Image#width / 2 —

    イメージの中心X座標

  • :center_y (Float) — default: Image#height / 2 —

    イメージの中心Y座標

  • :flip (Symbol) — default: Image#height / 2 —

    反転( horizontal: 横, vertival: 縦 )
