Overview

キーボード、ゲームパッド(コントローラー)、マウスからの入力情報を受け取ります。

※ 以下は一部メソッドで取得の際に必要なキーやボタン名のシンボルですので、使用の際の参考に。

key_sym =>  :_1, :_2, :_3, :_4, :_5, :_6, :_7, :_8, :_9, :_0
  :f1, :f2, :f3, :f4, :f5, :f6, :f7, :f8, :f9, :f10, :f11, :f12,
  :a, :b, :c, :d, :e, :f, :g, :h, :i, :j, :k, :l, :m, :n, :o, :p, :q, :r, :s, :t, :u, :v, :w, :x, :y, :z
  :enter, :escape, :backspace, :tab, :space, :minus, :equals, :leftbracket, :rightbracket, :backslash, :nonushash
  :semicolon, :apostrophe, :grave, :comma, :period, :slash, :capslock, :printscreen, :scrolllock, :pause, :insert
  :home, :pageup, :delete, :end, :pagedown, :up, :down, :left, :right, :numlockclear
  :kp_divide, :kp_multiply, :kp_minus, :kp_plus, :kp_enter, :kp_1, :kp_2, :kp_3, :kp_4, :kp_5, :kp_6, :kp_7, :kp_8, :kp_9, :kp_0
  :kp_period, :lctrl, :lshift, :lalt, :lgui, :rctrl, :rshift, :ralt, :rgui

pad_sym =>  :up, :down, :left, :right, _0, _1, _2...(ボタン数に応じて増加)

mouse_sym =>  :lbtn, :mbtn, :rbtn

リピートとは、押した瞬間の判定時間や、二回目以降の判定間隔に適用される値です。 
( #key_push?等、push?系メソッドの挙動を変えるもので、down?及びup?には影響はありません )

Overloads:

  • + (self) set_repeat(duration = 1, wait = -1, interval = 0, pad_index = -1)

    Parameters:

    • duration (Numeric) (defaults to: 1) —

      最初に押したと判定される持続時間

    • wait (Numeric) (defaults to: -1) —

      duration終了後、二回目の押された判定が行われるまでの間隔(待機時間)

    • interval (Numeric) (defaults to: 0) —

    • pad_index (Numeric) (defaults to: -1) —
